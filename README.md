## Overview

Beluga is an open-source cybersecurity laboratory tool designed for:

* Ethical security testing
* OSINT research
* Network analysis
* Automation scripting
* Defensive cybersecurity learning

Use only on systems you own or have explicit permission to test.

**Version:** 1.0 Beta

## Installation

### Clone Repository

```bash
git clone https://github.com/EgooDev/beluga-tools
cd beluga
```

### Create Virtual Environment (Recommended)

Linux / Mac:

```bash
python3 -m venv venv
source venv/bin/activate
```

Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Launch Beluga

```bash
python beluga.py
```
